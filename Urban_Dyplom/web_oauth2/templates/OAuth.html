Данные полученные при регистрации приложения на yandex

client_id : {{client_id}}

client_secret: {{client_secret}}

Далее делаем GET запрос по адресу https://oauth.yandex.ru/authorize?response_type=code&client_id=ПОДСТАВЛЕМ CLIENT_ID

В ответ получаем код авторизации (access_code): {{code}}

Далее делаем POST запрос по адресу https://oauth.yandex.ru/token и передаем в парметрах

'grant_type': 'authorization_code',

'code': code,

'client_id': client_id,

'client_secret': client_secret,

'redirect_uri': 'http://localhost:8000/accounts/yandex/login/callback.html/', }

получаем access_token : {{access_token}}

После этого можно запросить авторизационные данные по адресу https://login.yandex.ru/info
